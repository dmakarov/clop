Overview

CLOP is a domain specific language for separating an algorithm from the device specific optimizations that improve the performance of the algorithm on the device. The CLOP implementation uses the D programming language as its host language. The CLOP syntax is similar to, but not restricted by the syntax of D. CLOP fragments embedded in the application code allow to refer to the objects defined in the host application. 

/++
 + Compute the product of two matrices using CLOP.
 +/
void clop_mul_matrices()
{
  mixin (compile(q{
        NDRange(i : 0 .. size, j : 0 .. size)
        {
          int k;
          R[i, j] = 0;
          for (k = 0; k < size; ++k)
            R[i, j] += M[i, k] * N[k, j];
        }
      }));
}
